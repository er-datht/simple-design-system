# Monorepo Conversion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert the single-package Vite + React project into a Yarn workspaces monorepo with two independent apps (`web` and `admin`) under `srcs/`.

**Architecture:** Move the entire current project into `srcs/web/` with no code changes. Create a minimal Vite + React + TypeScript + Tailwind scaffold at `srcs/admin/`. Add a root `package.json` with Yarn workspaces pointing at `srcs/*`.

**Tech Stack:** Yarn workspaces, Vite 7, React 19, TypeScript 5.9, Tailwind CSS 4

**Spec:** `docs/superpowers/specs/2026-04-21-monorepo-conversion-design.md`
